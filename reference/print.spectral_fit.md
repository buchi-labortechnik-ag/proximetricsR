# Print method for an object of class `spectral_fit`

Prints the content of an object of class `spectral_fit`

## Usage

``` r
# S3 method for class 'spectral_fit'
print(x, ...)
```

## Arguments

- x:

  an object of class `spectral_fit` (as returned by the
  [`.estimate_model`](https://buchi-labortechnik-ag.github.io/proximetricsR/reference/dot-estimate_model.md)
  function).

- ...:

  arguments to be passed to methods (not functional).

## Value

Returns `x` invisibly.

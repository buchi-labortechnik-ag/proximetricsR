# Read ProxiMate (.tsv) files

This function imports .tsv files generated by BUCHI ProxiMate sensors.

## Usage

``` r
proximate_read_data(file)
```

## Arguments

- file:

  A string indicating the name (and path) of the .tsv file. A
  [`textConnection`](https://rdrr.io/r/base/textconnections.html) can
  also be passed.

## Value

A data.frame containing all the metadata, response variables and spectra
in the tsv file. The spectra is returned in a matrix embedded in the
data.frame which can be accessed as `...$spc`.

## Examples

``` r
data("NIRcannabis")
filename <- paste0(tempdir(), "/NIRcannabis.tsv")
# Need to produce a tsv file before we can read it
proximate_write_data(
  x = NIRcannabis,
  file = filename,
  properties = c("CBDA", "THCA", "CBD", "THC")
)
# Equivalent to dataset NIRcannabis
dat <- proximate_read_data(filename)
```

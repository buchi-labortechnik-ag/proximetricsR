# Summary of `spectral_model`

Gets a summary of an object of class `spectral_model`

## Usage

``` r
get_model_summary(x, ...)
```

## Arguments

- x:

  an object of class `spectral_model` (as returned by the
  [`calibrate`](https://buchi-labortechnik-ag.github.io/proximetricsR/reference/calibrate.md)
  function).

- ...:

  arguments to be passed to methods (not functional).

## Value

A list containing a summary of the model.
